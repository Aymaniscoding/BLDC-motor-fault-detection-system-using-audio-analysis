# Signal & Image Processing Project

## Overview
This project focuses on Signal and Image Processing using Deep Learning techniques for BLDC motor sound classification. The system processes audio signals, extracts frequency-domain features, generates Mel Spectrograms, and applies Convolutional Neural Networks (CNNs) for condition detection and classification.

## Features
- Audio signal preprocessing
- Fourier Transform feature extraction
- Mel Spectrogram visualization
- Deep learning-based classification
- CNN model implementation using PyTorch
- Data visualization and analysis

## Technologies Used
- Python
- PyTorch
- NumPy
- Librosa
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Load and preprocess audio data
2. Convert signals into spectrogram representations
3. Extract important frequency features
4. Train CNN model on processed data
5. Evaluate model performance and visualize results
